一些關於 Mapping 的小事情

  1. 扁不扁平看開發需求
  2. 需不需要做 tokenized 看搜尋需求
    1. name, code, position 需要被分析嗎?
  3. 搜尋先覺是否都用不到 Nested Mapping?
    1. Nested Object 內的屬性是有關聯的
    2. 例子: 把姓氏跟名子分開存時, 兩者存在強烈關聯
    3. 王思平 !== 陳思平
    4. http://joelabrahamsson.com/elasticsearch-nested-mapping-and-filter/
